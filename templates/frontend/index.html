Golang

如何学习好golang

可以通过阅读官方文档、教程、书籍和视频来学习。建议学习变量、函数、控制流、数据类型等基本语法和概念。从官方文档开始。
Gin框架

gin框架是golang中比较常见的web框架

Gin是一个流行的Golang Web框架，它提供了优雅的封装和友好的API。Gin在Golang的HTTP标准库net/http之上进行了封装，提供了更强大的路由解析能力和更方便的中间件操作。
go-zero微服务

go-zero是一个基于Go语言的Web和微服务开发框架

go-zero 是一个集成了各种工程实践的 web 和 rpc 框架。通过弹性设计保障了大并发服务端的稳定性,经受了充分的实战检验。
MySQL数据库

MySQL是一个关系型数据库管理系统

MySQL是一种关联性数据库,将数据保存在不同的表中,而不是将所有的数据保存在一个大仓库内,增加了速度,提高了灵活度,MySQL提供开源版本,不收取费用
Redis缓存

Redis 是一个高性能的key-value数据库

Redis是一个开源的内存中数据结构存储系统，它支持多种类型的数据结构，如字符串(strings)、散列(hashes)、列表(lists)、集合(sets)和有序集合(sorted sets)。Redis以其高性能、高并发和丰富的数据结构而闻名，广泛应用于缓存、消息队列、排行榜、社交网络等多种场景。
Grpc远程过程调用

gRPC 是一个高性能、开源和通用的 RPC 框架,面向服务端和移动端,基于 HTTP/2 设计。

gRPC中的 metadata 类似于我们在 HTTP headers中的键值对,元数据可以包含认证token、请求标识和监控标签等
